## First Principio Group (December 2014 ΟΔβ²§βÄ€ December 2018)

### Context & Motivation

**Why Entrepreneurship:**

- Worked with many app developers at Microsoft (large corporations to small indie teams)
- Always had passion for entrepreneurship
- Inspired to build own application
- Financial runway: "flush with cash" from bonuses after two very successful Microsoft years

**Decision:** Resigned from Microsoft in November 2014 to pursue entrepreneurial aspirations.

### Learning Phase

Immersed in the intricacies of starting a business:

- Legal requirements
- Infrastructure setup
- Market analysis
- Product management
- App development
- Marketing
- Analytics-based refinement

### Product: WotCom (World of Tanks Companion App)

**Product Vision:**

- Companion app for World of Tanks, a popular European game
- Gap in market: No Windows companion app existed
- Target: Windows ecosystem users in gaming community

**Technical Architecture:**

| Component    | Technology          | Purpose                                                                            |
| ------------ | ------------------- | ---------------------------------------------------------------------------------- |
| Design       | Metro UI guidelines | Windows 8 design language; what Microsoft considered a "great Windows application" |
| Cloud - Auth | Amazon Cognito      | User accounts and signups                                                          |
| Cloud - Data | Amazon RDS          | SQL database for app data                                                          |
| Development  | Xamarin             | Cross-platform framework; goal to port to iOS and Android                          |

**Team & Development Process:**

| Aspect       | Detail                                                  |
| ------------ | ------------------------------------------------------- |
| Team size    | 2 developers in Ukraine                                 |
| Cadence      | Daily morning meetings to review progress               |
| Robin's role | Product management, testing, cloud services development |
| Approach     | Continuous development for speed to market              |
| Timeline     | **3 months** from start to first version launch         |

### Go-to-Market

**Strategy:** Scrappy social media marketing campaign targeting the gaming community

**Results:**

| Metric           | Result                                                 |
| ---------------- | ------------------------------------------------------ |
| Category ranking | **Top 10** in lifestyle category                       |
| Geographic reach | 4 countries in top 10                                  |
| Major markets    | **Top 200** in US, Britain, France, Germany, Australia |

### Why It Ended

| Factor             | Detail                                                  |
| ------------------ | ------------------------------------------------------- |
| User engagement    | Not enough active usage to warrant continued investment |
| Platform challenge | Windows Store apps notoriously underperformed           |
| Revenue            | Not generating income quickly enough                    |
| Runway             | Ran out of money set aside for business                 |
| Opportunity cost   | Verizon approached with offer (September 2015)          |

**Timeline:**

- September 2015: Scaled back to single developer for maintenance
- December 2018: Shut down application and closed business

### Key Learnings

"Very rewarding experience and helped me grow both personally and in my career."

- Platform selection matters (Windows Store underperformance)
- User engagement > vanity metrics (app store rankings)
- Cash runway planning for consumer apps

## Second Principio Group (April 2023 ΟΔβ²§βÄ€ Present)

### Catalyst: The ChatGPT Moment

**Context:** Left Google Cloud in March 2023 (layoff as part of 12,000-employee reduction).

**Realization:**

> "When ChatGPT launched, I realized that large language models were the missing pieces of the puzzle to unlock the full potential of the work that I was doing around leveraging AI to generate audio and video content."

**Assessment:**

- LLMs = "game changer"
- "Most disruptive technologies of our time"
- "Will change the way we work in ways we have not fully grasped yet"

**Strategic Imperative:**

> "I needed to get ahead of this and reskill myself for the future world dominated by artificial intelligence and really understand how LLMs work, as well as artificial intelligence more broadly."

### Reskilling Journey

**Python:**

- Recognized as "lingua franca for developing AI applications"
- Set about learning Python programming

**Generative AI Exploration:**

- Intrigued by MidJourney (generative art)
- "I learn best by applying my newly developed skills"

### Hands-On Learning Project: Etsy Art Store

**Purpose:** Learn Python + get hands-on OpenAI API experience

**Application Built:** MidJourney artwork management system for Etsy sales

| Component        | Technology                 | Function                                     |
| ---------------- | -------------------------- | -------------------------------------------- |
| Image processing | ActionScript + Photoshop   | Automate resizing artwork                    |
| Descriptions     | OpenAI API + GPT-3.5-turbo | Generate SEO-optimized store listings        |
| Listing upload   | Etsy API                   | Automated store listing creation             |

| Learning tool    | ChatGPT                    | Learn Python, Firebase deployment, API calls |

**Outcome:** "Fun way to learn a new programming language and get hands-on experience using the OpenAI API."

### Business Exploration

**Goal:** Develop passive income stream

**Opportunities Explored:**

| Partner | Opportunity                             | Assessment         |
| ------- | --------------------------------------- | ------------------ |
| SADA    | Value engineering consultant (contract) | Not passive income |

**Pivot:**

> "Got to thinking about how we could use artificial intelligence to provide a SaaS based solution for value engineering and sales operations."

**Action:** Connected with former Google Cloud colleagues to explore business leveraging generative AI opportunity.
