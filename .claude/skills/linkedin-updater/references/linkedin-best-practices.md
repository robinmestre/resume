# LinkedIn Best Practices Reference

## Section Specifications

### Headline (220 characters max)

**Formula:** `[Role] | [Specialization] | [Quantified Value] | [Key Differentiator]`

**Why This Works:**
- Includes primary job title (searchable)
- Shows specialization area (differentiates)
- Quantifies impact (proves credibility)
- Highlights unique value (memorable)

**AI Expert Keywords to Include:**
- Primary: Generative AI, AI Strategy, AI Architecture
- Secondary: LLMs, AI Platform, Enterprise AI
- Trending: Agentic AI, Multi-Agent Systems

**Example Headlines:**
```
Principal AI Architect | Zero-to-One Program Builder | $1.5B Pipeline | 100% Program Survival Rate
```
```
AI Platform Architect | Permanence Architecture Expert | 6 Programs Running 3-7+ Years Post-Departure
```

### About Section (2600 characters max)

**Five-Part Structure:**

| Part | Purpose | Length | Source |
|------|---------|--------|--------|
| 1. Hook | Bold claim visible before "see more" | 1-2 sentences | Elevator pitch |
| 2. Who You Help | Target audience + transformation | 2-3 sentences | Positioning |
| 3. Proof Points | Quantified achievements | 3-5 bullets | Key metrics |
| 4. Philosophy | Unique methodology/lens | 1-2 sentences | Differentiators |
| 5. CTA | Connection invitation | 1 sentence | Custom |

**Writing Guidelines:**
- First-person voice ("I architect..." not "Robin architects...")
- 8th grade reading level (Hemingway score)
- No jargon or hyperbole
- Professional, direct, quietly confident tone

**Hook Examples:**
- "I architect programs for permanence."
- "Every program I've built across three companies is still running."
- "The Stratozone program I led became Google Cloud Migration Center."

**Placement Priority:**

| Location | SEO Weight | Strategy |
|----------|------------|----------|
| Headline | Highest | Top 2-3 role keywords |

| Current Job Title | High | Exact match to target roles |
| Skills Section | High | All searchable terms |

| Tier | Examples | Placement |

### Experience Section

**Format:** CAR (Challenge-Action-Result)

**Conversion from STAR:**
- Challenge = Situation + Task (context)
- Action = Action (what you did)
- Result = Result (quantified outcome)

**Bullet Template:**
```
[Action verb] [what was done] to [address challenge], resulting in [quantified result]
```

**Guidelines:**
- Lead with outcomes, not responsibilities
- 3-5 bullets per role
- Quantify everything possible
- Use 8th grade English
- Vary action verbs

**Power Verbs by Category:**

| Category | Verbs |
|----------|-------|
| **Leadership** | Directed, Led, Orchestrated, Spearheaded, Championed |
| **Building** | Architected, Built, Created, Designed, Developed, Established |
| **Growth** | Scaled, Expanded, Grew, Accelerated, Amplified |
| **Optimization** | Streamlined, Optimized, Transformed, Modernized, Consolidated |
| **Delivery** | Delivered, Launched, Shipped, Deployed, Executed |
| **Influence** | Secured, Won, Convinced, Negotiated, Partnered |

**Verbs to Avoid:**
- Responsible for (passive)
- Helped (weak)
- Worked on (vague)
- Assisted (subordinate)

### Featured Section

**Recommended Content Types:**

1. **Case Studies** — Demonstrating methodology (Permanence Architecture)
2. **Speaking Engagements** — Thought leadership visibility
3. **Certifications** — Active credentials (Six Sigma Black Belt)
4. **External Validation** — Articles, awards, mentions
5. **Portfolio Items** — Architecture artifacts (if shareable)

**Priority Order:**
1. Content that demonstrates unique methodology
2. External validation (third-party credibility)
3. Certifications and credentials

## Quantification Patterns

**Revenue/Pipeline:**
- "Generated $X pipeline in Y months"
- "Scaled from $X to $Y (ZX growth)"

**Efficiency:**
- "Reduced X by Y%"
- "Improved X by Y%"
- "Achieved YX completion rate vs baseline"

**Scale:**
- "Led X contributors across Y workstreams"
- "Trained X+ participants"
- "Created X+ artifacts"

**Longevity:**
- "Program still running X years post-departure"
- "X programs across Y companies, all still operational"
